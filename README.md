📖 About The Project

Welcome to my **personal website** — a space where I share my journey as a **Web Developer**, **Student**, and **Author**.

This project is more than just a portfolio. It's a reflection of who I am — my **projects**, my **friends**, my **story**, and my **vision**. Every page is built with ❤️ and attention to detail.

> 💡 *"Somewhere Between I Want It & I Got It"* — Ravi Raj

✨ Features

- 🏠 **Portfolio** — Showcase of projects, skills & milestones
- 📖 **Autobiography** — 12 chapters in English & Hinglish + E-book download
- 👥 **Friends Corner** — 14 friends with voice search, filters & PDF friend cards  
- 💬 **AI Chatbox (RJ AI)** — Virtual assistant with 20+ intent detection
- 📄 **Resume** — Professional CV with print & download support
- ✉️ **Contact Form** — EmailJS-powered messaging, no backend needed
- 🌓 **Theme Toggle** — Dark/Light mode with localStorage
- 📱 **Fully Responsive** — Mobile-first design
- ⚡ **Fast & Clean** — Optimized assets, modular code

🚀 How to Run Locally

1. **Clone the repository**
git clone https://github.com/ravirajhere/ravirajhere.github.io.git
2. *Navigate to the folder*
cd ravirajhere.github.io
3. *Open in browser*
   - Double-click `index.html` 
   - OR use `Live Server` in VS Code

4. *No backend required* - Everything runs client-side
