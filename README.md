📖 About The Project

This is my personal website — a reflection of who I am as a **Web Developer, Student, and Author**.

Here you’ll find my journey, projects, story, friends, and the tools I’ve built to learn and grow.

> *"Somewhere Between I Want It & I Got It"* — Ravi Raj

---

✨ Features

- 🏠 **Portfolio** — Showcase of projects, skills & milestones
- 📖 **Autobiography** — 12 chapters in English + Hinglish toggle + E-book download
- 👥 **Friends Corner** — 14 friends with voice search, filters & PDF friend cards
- 💬 **AI Chatbox (RJ AI)** — Virtual assistant with 20+ intents, dark/light mode
- 📄 **Professional Resume** — Printable CV with jsPDF + html2canvas
- ✉️ **Contact Form** — EmailJS powered messaging, no backend needed
- 🌓 **Dark/Light Mode** — Theme toggle with localStorage
- 📱 **Fully Responsive** — Mobile first design, works on all devices
- ⚡ **Fast & Clean** — Optimized CSS/JS, modular file structure

🚀 How to Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/ravirajhere/ravirajhere.github.io.git
2. *Navigate to the folder*
   cd ravirajhere.github.io
3. *Open in browser*
   - Double-click `index.html` 
   - OR use `Live Server` extension in VS Code

4. *Start exploring*
   All pages are interlinked from the homepage
