Hello! I'm Ravi Raj

A passionate Student and 12th-grade student from Bandwar, Begusarai, Bihar. My coding journey started in 2023, and since then, I've been obsessed with creating beautiful, functional websites.

I started with basic HTML and CSS, and now I'm diving deep into JavaScript and Python. Every day I challenge myself to learn something new. I believe in learning by building — that's why I create projects constantly to improve my skills.
